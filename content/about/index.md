# À propos

Bienvenue sur MonLabAzure.

Je suis ingénieur infrastructure et cloud avec plus de 15 ans d'expérience dans l'administration système, la production informatique, la virtualisation et les environnements Cloud.

Au cours de ma carrière, j'ai accompagné des projets de transformation et d'industrialisation autour de :

* Microsoft Azure
* Sécurité et gouvernance Cloud
* Terraform et Infrastructure as Code
* SQL Server
* Active Directory
* VMware
* Linux
* Automatisation PowerShell


Ce blog a été créé pour partager des retours d'expérience, des travaux pratiques, des architectures de référence, des ADR (Architecture Decision Records) et des guides techniques exploitables en environnement réel.

L'objectif est de documenter des solutions pragmatiques, maintenables et alignées avec les bonnes pratiques de production.

## Thématiques principales

* Architecture Azure
* Gouvernance Cloud et FinOps
* Réseau et sécurité
* Terraform
* SQL Server
* Active Directory
* Linux
* DevOps et automatisation

## Philosophie

Privilégier la simplicité, l'automatisation, la documentation et la reproductibilité.

Chaque article publié sur MonLabAzure vise à fournir une valeur opérationnelle directement réutilisable sur le terrain.
