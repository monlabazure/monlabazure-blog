# À propos

Je suis ingénieur Infrastructure et Cloud avec plus de 15 ans d'expérience dans la conception, l'exploitation et l'automatisation d'environnements de production.

Mes domaines d'expertise couvrent :

* Microsoft Azure
* Architecture Cloud
* Terraform
* SQL Server
* Active Directory
* Linux
* VMware
* Automatisation PowerShell

MonLabAzure a pour objectif de partager :

* des ADR (Architecture Decision Records)
* des architectures de référence
* des guides techniques
* des retours d'expérience
* des bonnes pratiques de production

L'objectif est de fournir un contenu pragmatique, directement exploitable dans un contexte professionnel.
